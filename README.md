# brownian-motion-simulation
2D Brownian Motion simulation using Monte Carlo method in Python

Description
This project simulates the random motion of particles in two dimensions using the Monte Carlo method. It demonstrates Brownian motion and diffusion behavior.

Objective
To model stochastic motion of particles and visualize their random trajectories using Python.

Method
- Random steps are generated using normal distribution
- Positions are calculated using cumulative sum
- Multiple particles are simulated

Result
The simulation shows random paths of particles starting from an initial point and spreading over time.

Tools Used
- Python
- NumPy
- Matplotlib
